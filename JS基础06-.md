**null 和 undefined 属于两种不同类型，使用全等运算符（===）或 typeof 运算符可以进行检测。**

```javascript
NaN === NaN //false
NaN !== NaN //true
```

null 和 undefined 都表示空缺的值，转化为布尔值时都是假值，可以相等。

```javascript
console.log(null == undefined);  //返回 true
```

如果直接在表达式中使用未声明的变量 b ，会引发异常。

```javascript
console.log(b == undefined); //b未定义的错误信息
```
